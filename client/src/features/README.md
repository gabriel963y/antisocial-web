# features

Módulos por funcionalidad (domain-driven). Cada carpeta agrupa componentes, hooks y lógica de una feature específica.

**Carpetas:**
- `auth/` — Login y Register
- `comments/` — CommentList y CommentForm
- `posts/` — Feed, PostDetail, CreatePost
- `profile/` — Profile del usuario logueado
- `users/` — (opcional) listado y búsqueda de usuarios
