# stores

Estados globales con Zustand.

El enunciado pide usar `useContext` para el usuario logueado, pero si prefieren usar Zustand para otros estados globales pueden hacerlo acá.

**Ideas (opcionales):**
- `useAuthStore.ts` — manejo de usuario logueado en lugar de Context (alternativa)
- `useUIStore.ts` — estado de UI (sidebar abierta, tema, toasts, etc.)
