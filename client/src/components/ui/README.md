# components/ui

Componentes atómicos reutilizables. Si ven que repiten mucho markup (botones, inputs, cards).

**Ejemplo :**
- `Button.tsx` — botón reutilizable con variantes (primary, secondary, danger)
- `Input.tsx` — input controlado con label y mensaje de error
- `Card.tsx` — contenedor tipo tarjeta para posts
- `ProtectedRoute.tsx` — componente que envuelve rutas y redirige a /login si no hay usuario logueado
- `Spinner.tsx` — indicador de carga
- `TagBadge.tsx` — badge para mostrar una etiqueta (tag)
